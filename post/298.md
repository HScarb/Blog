# C++字符串按字母顺序排序算法

```C++
string strSort(string &s)
{
    int count[26] = { 0 };
    int n = s.size();
    // 记录每个字符的个数
    for (int i = 0; i < n; ++i)
    {
        count[s[i] - 'a']++;
    }
    int p = 0;
    // 长度为n的a字符串t
    string t(n, 'a');
    // 遍历26个字符，如果这个字符有count，那就放入count个该字符进t
    for (int i = 0; i < 26; ++i)
        for (int j = 0; j < count[i]; ++j)
            t[p++] += i;
    return t;
}
```
